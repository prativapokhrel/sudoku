# SUDOKU
A simple 9x9 sudoku game using Ruby 

### Running
To run the project locally, install the dependencies using Bundler
```
bundle install
```

Then, play the game by running
```
bundle exec ruby game.rb
```

### Tests
RSpec has been used for writing tests 
```
bundle exec rspec 
```

### Linter/Formatter
```
bundle exec rubocop 
```
